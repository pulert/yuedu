### YueDu
阅读3.0, 阅读是一款可以自定义来源阅读网络内容的工具，为广大网络文学爱好者提供一种方便、快捷舒适的试读体验。


### 介绍
书源、订阅号、规则相关.

### 使用说明
安装新版阅读,打开订阅——规则订阅——点击右上加号——按类型粘贴以下地址即可.
